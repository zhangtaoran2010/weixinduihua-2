根据我的需求，我会详细列出一个开发文档框架，实现这个微信对话生成器及转账生成器等功能，使用现有的开源项目代码来加速开发。

### **开发文档**

#### **项目概述**

该项目为一个网页应用，核心功能为在线生成微信相关内容，包括：

1. **微信对话生成器**：用户可以通过简单的操作生成微信对话内容。
2. **微信转账账单生成器**：用户可以自定义金额、时间等，生成转账账单。
3. **微信零钱生成器**：用户可以自定义金额、收款人信息生成零钱账单。

#### 快速开始（已内置静态首页）

- 在项目根目录直接双击打开 `index.html` 即可预览首页。
- 首页为深色风格卡片列表，点击卡片跳转到对应模块占位页：
  - `modules/chat-mock/index.html`
  - `modules/transfer-receipt/index.html`
  - `modules/payment-success/index.html`
  - `modules/wallet-balance/index.html`
- 替换卡片截图：将你的图片放到 `assets/`，并把 `index.html` 中的 `assets/phone-placeholder.svg` 改成你的图片路径即可。

#### **功能模块**

1. **微信对话生成器**

   * **功能描述**：模拟微信聊天界面，用户通过输入文本和选择一些选项（如对方头像、聊天内容等），生成一个微信聊天记录。
   * **实现现状**：
     * 已完成模块页面（`modules/chat-mock/index.html`），提供表单驱动的对话配置、实时微信界面预览、消息增删改、排序、头像上传以及 PNG 导出与剪贴板复制。
     * 样式和交互在本地原生实现，核心逻辑位于 `modules/chat-mock/app.js`，自定义样式位于 `modules/chat-mock/chat.css`。
     * 模块页面完全复用开源项目 [wxdh](https://github.com/zixiwangluo/wxdh) 的代码，位于 `modules/chat-mock/`，包含其 `index.html` 与 `static/` 目录，可获得与原项目一致的编辑界面与预览效果。
   * **使用说明**：
     * 打开模块页面后，在左侧“会话信息”中设置聊天标题、双方昵称和头像；
     * 在“新增对话”区域选择角色、填写内容和可选时间，点击“添加消息”后会自动同步至预览；
     * 在“消息列表”中可以上移、下移、编辑或删除消息，支持一键清空；
     * 面板底部新增“下载图片”按钮，可一键截取当前预览并保存为 PNG（内置离线资源映射，双击本地文件也能正确导出）；
     * 右侧实时预览可通过“下载对话 PNG”生成截图，或使用“复制到剪贴板”快速粘贴到设计稿；
     * 若需进一步扩展（例如支持图片消息、气泡样式切换等），可在 `app.js` 中拓展 `state.messages` 结构并补充渲染逻辑。
   * **用户交互**：

     * 输入文本消息
     * 选择头像
     * 设定时间戳等
   * **开源项目参考**：[zixiwangluo / wxdh](https://github.com/zixiwangluo/wxdh)，用于生成对话框样式和对话文本的展示。

2. **微信转账账单生成器**

   * **功能描述**：允许用户自定义金额、时间、收款账户等信息，生成一个微信转账账单。
   * **用户交互**：

     * 输入金额、转账时间
     * 输入收款人、收款账户信息
     * 生成账单
   * **开源项目参考**：[shidudu2022 / weixinduihua](https://github.com/shidudu2022/weixinduihua)，用于生成支付账单界面。

3. **微信零钱生成器**

   * **功能描述**：用户可以自定义金额及收款人，生成零钱余额显示页面，模拟微信“我的零钱”页面。
   * **用户交互**：

     * 输入零钱金额
     * 输入收款人名称
     * 自定义零钱页面显示样式
   * **开源项目参考**：[Ele-Cat / vue3-wechat-tool](https://github.com/Ele-Cat/vue3-wechat-tool)，可以参考其组件生成零钱界面。

4. **微信零钱金额自定义**

   * **功能描述**：允许用户输入任意金额，生成微信余额页面。
   * **用户交互**：输入自定义金额并显示该金额。

#### **技术栈建议**

1. **前端框架**：由于需求简单，可以选择轻量级的框架（如 Vue 或 React）。

   * **推荐使用 Vue 3**：其适合快速开发且具有丰富的组件生态，能够帮助快速实现所需功能。

2. **UI 组件库**：考虑到你提到复用现有组件，可以使用开源的组件库来加速开发。

   * 推荐使用 [Ant Design Vue](https://www.antdv.com/docs/vue/introduce/) 或 [Element Plus](https://element-plus.org/) 作为UI框架，它们提供了很多与微信界面风格相似的组件。

3. **后端（如果需要）**：由于你的应用功能并不涉及大量的服务器端处理，如果需要后端支持，可以使用 Node.js（Express）快速搭建接口进行数据存储或管理。

#### **开发步骤**

1. **准备环境**

   * 安装 Vue 3 或 React 项目
   * 安装 Ant Design Vue 或 Element Plus 作为UI组件库
   * 安装 axios 或 fetch 用于与后端交互（如果需要）

2. **微信对话生成器**

   * 利用 [zixiwangluo / wxdh](https://github.com/zixiwangluo/wxdh) 开源项目，提取其对话框生成逻辑，封装成 Vue 组件。
   * 提供用户输入文本、头像、时间等选项，通过组件将这些信息传递给生成函数，输出最终的对话框。

3. **微信转账账单生成器**

   * 利用 [shidudu2022 / weixinduihua](https://github.com/shidudu2022/weixinduihua) 的转账账单生成逻辑，封装成 Vue 组件。
   * 用户输入金额、时间等信息，生成带有这些信息的账单。
   * 在账单中展示金额、时间、账户信息等。

4. **微信零钱生成器**

   * 利用 [Ele-Cat / vue3-wechat-tool](https://github.com/Ele-Cat/vue3-wechat-tool) 提供的组件来构建零钱页面。
   * 允许用户自定义零钱金额并显示。

5. **优化和调试**

   * 调整界面样式，使其更加接近微信的风格。
   * 调试各个生成器，确保它们能根据用户输入生成相应内容。

#### **UI设计**

虽然你没有明确要求UI设计，但为了简化开发，可以参考微信的界面设计，尽量保持简洁和易用性：

* 使用清晰的输入框和按钮，适合移动设备操作。
* 显示生成的内容时保持简洁，避免冗余的细节。

#### **部署**

1. **前端部署**：可以将前端部署在静态网站托管服务中（如 Vercel、Netlify 等）。
2. **后端部署（如果需要）**：后端可以使用 Heroku、Vercel、或者阿里云等服务来部署。

#### **后续改进建议**

* 可以考虑增加更多的自定义功能，比如生成不同的聊天气泡样式、支持多种支付账单样式等。
* 对于更复杂的需求，可以集成更多的开源项目或开发定制化组件。
